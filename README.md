# 节奏之王 · The King of Rhythm

用浏览器原生技术（Canvas + Web Audio API）实现的节奏天国（Rhythm Heaven）风格小游戏。
零依赖、无构建步骤，所有音乐与音效均为实时合成，无需任何音频文件。

## 运行

直接双击打开 `index.html` 即可游玩；或起一个本地服务器：

```bash
python3 -m http.server 8000
# 浏览器打开 http://localhost:8000
```

> 首次点击页面才会开启声音（浏览器自动播放策略），请调高音量。

## 玩法

- **三模式**：每关可选 简单 / 普通 / 困难。简单=基础谱面；普通=时长 +40~60%、BPM ×1.1、新增花样；困难=BPM ×1.2、密度更高且**种子随机**（每次开局节奏都不同）
- **主键**：`空格` / `回车` / `J` / 点击画面 —— 击打、踏步、灌油、啄米、昂首
- **副键**：`F` / `K` —— 仅双键关卡使用（第 14 关拍虫子）
- **长按**：灌油/合唱/冲刺/DJ/重拳等关中按住主键蓄力、听信号松开
- `Esc` = 退出到选关
- **判定**：±50ms 内 PERFECT，±120ms 内 GOOD，超时未按为 MISS，按在空处会"挥空"断连击
- **评级**：按命中率结算 S / A / B / C，各关各模式的最佳评级分别记录
- **移动端**：进入游戏自动尝试全屏并锁定横屏；浏览器不支持横屏时给出选择——「强制横屏显示」（CSS 90° 旋转画面，任何浏览器都有效）或「竖屏小屏继续玩」；点击屏幕=主键，按住=长按（pointerId 跟踪，含 iOS 长按 callout 处理）；双键关卡左半屏=F、右半屏=空格（强制横屏模式下自动按旋转后坐标映射）

### 设计理念：音频即谱面

参照节奏天国原作（[Blue Birds](https://rhwiki.net/wiki/Blue_Birds)、[Lockstep](https://rhythmheaven.fandom.com/wiki/Lockstep)、[Rhythm Rally](https://www.bilibili.com/read/cv5771291/)）：

- **指令是唱出来/听出来的**：口令、脚步声、球声都嵌在音乐里并提前固定时间给出
- **回应是学会的固定节奏型**：例如蓝鸟的「突突突」→ 三连啄（半拍一下）
- **按对时你的声音融入音乐，按错时产生可闻的不和谐**；画面只负责氛围与确认，蒙眼也能玩

## 技术要点

- **时钟源**：`AudioContext.currentTime` 是唯一时钟。音乐排程（`setInterval` 25ms + 0.15s lookahead）
  与按键判定都从它推导，避免 `setTimeout`/`rAF` 直接计音乐时间带来的漂移（见 `js/conductor.js`、`js/game.js`）
- **合成音频**：鼓组、贝斯、旋律、口令、音效全部由 `Oscillator` + 噪声 buffer 实时生成（`js/audio.js`）
- **关卡结构**：每关声明 BPM 与音符表（第几拍需要按键），渲染只读时钟、逻辑只排事件。
  扩展机制：音符带 `key` 字段启用双键判定，带 `dur` 字段启用长按判定（`js/game.js`）
- **绘制**：所有角色/场景用 Canvas 几何图形手绘，`Draw.blob` 支持四肢/表情/配饰（`js/levels.js`）

## 人工试听检查

自动化测试无法验证"音乐是否合拍"。自检方法：进入第 1 关，听鼓点是否与物品到达圆圈的瞬间一致；
若你的设备音频延迟较大（蓝牙音箱尤其明显），会感觉判定整体偏早/偏晚——本项目暂未做延迟校准，
有线/扬声器输出体验最佳。


> 新增关卡约定：「大团圆 Remix」永远是最后一关。新关卡文件在自身末尾用
